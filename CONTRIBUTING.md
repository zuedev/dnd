# Contributing

This project is incredibly personal to me, and I don't expect anyone to contribute to it. However, if you do want to contribute, I'm more than happy to accept merge requests. I only ask that you follow the guidelines below.

## Issues

If you find a bug or have a suggestion, please open an issue. I'll try to respond as quickly as possible.

## Merge Requests

If you want to contribute to this project, please follow these guidelines:

- Do **NOT** touch anything regarding lore, worldbuilding, or anything else that is not code. I'm happy to accept suggestions, but I will not accept any changes to the actual content that is not in my own words.
- **Always** open an issue before starting work on a merge request and link them together. This will allow us to discuss the changes before you spend time working on them. The issue will be used to discuss the changes, and will be referenced in the merge request. The merge request's comments will be used to discuss the implementation of the changes specifically.
